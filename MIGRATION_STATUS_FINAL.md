|----------------|---------------|--------------|
| `selectSettingsOfEss` | `useSettingsStore` | Settings state, interface mode, buttons |

| `selectSettingsOfTgsTes` | `useTgsSettingsStore` | TGS/TES settings |

| `selectEssSwitch` | `useEssSwitchStore` | ESS switch data |
| `selectFaults` | `useFaultsStore` | Fault management |
| `selectForceAndCommand` | `useForceAndCommandStore` | Temperature control |
| `selectChartState` | `useChartStore` | Chart data |
| `selectTimerState` | `useTimerStore` | Timer state |
| `selectHeaterStatus` / `ssrDescriptionSlice` | `useHeaterStatusStore` | Heater/SSR status |
| `selectSystemIdentification` | `useSystemIdentificationStore` | System ID |
| `selectAdminSettings` | `useAdminSettingsStore` | Admin settings |

### Migration Steps for Each File

1. **Replace Redux Imports**
   ```javascript
   // Before
   import { useDispatch, useSelector } from 'react-redux';
   import { selectSettingsOfEss, setInterfaceMode } from '../store/slices/settingsOfEssSlice';

   // After
   import { useSettingsStore } from '../store/zustand';
   ```

2. **Replace useSelector Calls**
   ```javascript
   // Before
   const state = useSelector(selectSettingsOfEss);
   const mode = state.interfaceMode;
   const editButton = state.buttonsOfSettings.settingsEditButton;

   // After
   const mode = useSettingsStore((state) => state.interfaceMode);
   const editButton = useSettingsStore((state) => state.buttonsOfSettings.settingsEditButton);
   ```

3. **Replace dispatch Calls**
   ```javascript
   // Before
   const dispatch = useDispatch();
   dispatch(setInterfaceMode(true));
   dispatch(setSettingsEditButton());

   // After
   const setInterfaceMode = useSettingsStore((state) => state.setInterfaceMode);
   const setSettingsEditButton = useSettingsStore((state) => state.setSettingsEditButton);
   setInterfaceMode(true);
   setSettingsEditButton();
   ```

4. **Update useEffect Dependencies**
   ```javascript
   // Before
   useEffect(() => {
     dispatch(someAction());
   }, [dispatch]);

   // After
   useEffect(() => {
     someAction();
   }, [someAction]);
   ```
